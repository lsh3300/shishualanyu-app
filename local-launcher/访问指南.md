# 世说蓝语 - 应用访问指南

## 应用信息
- 应用名称: 世说蓝语 - 传承蓝染文化的数字平台
- 线上正式访问地址: https://shishualanyu-dx63zno4k-lsh3300s-projects.vercel.app
- 本地开发访问地址: http://localhost:3000

## 如何访问应用

### 方法1: 直接在浏览器中访问（推荐线上环境）
1. 打开任意浏览器（Chrome、Firefox、Edge等）
2. 在地址栏输入: https://shishualanyu-dx63zno4k-lsh3300s-projects.vercel.app
3. 按回车键访问

### 方法2: 使用提供的文件
1. 双击 `app_entry.html` 文件，然后点击"进入应用"按钮
2. 双击 `launch_app.html` 文件，它会在2秒后自动跳转到 **线上应用**
3. 双击 `世说蓝语.url` 快捷方式文件（已指向线上正式地址）
4. 双击 `start_app.bat` 批处理文件（用于本地开发时启动 localhost，可选）

### 方法3: 使用命令行（本地开发调试时）
1. 打开命令提示符或PowerShell
2. 输入以下命令之一:
   ```
   start http://localhost:3000
   ```
   或
   ```
   powershell -Command "Start-Process http://localhost:3000"
   ```

## 注意事项
- 确保应用服务器正在运行（通过执行 `npm run dev` 命令）
- 如果无法访问，请检查防火墙设置
- 确保端口3000没有被其他程序占用
