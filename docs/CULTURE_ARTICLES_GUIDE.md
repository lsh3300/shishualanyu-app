## 🔧 数据库结构说明

### culture_articles 表

| 字段 | 类型 | 说明 |
|------|------|------|
| id | UUID | 主键 |
| slug | TEXT | URL 友好标识（唯一） |
| title | TEXT | 文章标题 |
| excerpt | TEXT | 摘要 |
| content | TEXT | 正文内容 |
| cover_image | TEXT | 封面图片 URL |
| category | TEXT | 分类 |
| tags | TEXT[] | 标签数组 |
| read_time | INTEGER | 阅读时间（分钟） |
| author | TEXT | 作者 |
| views | INTEGER | 浏览量 |
| likes | INTEGER | 点赞数 |
| status | TEXT | 状态（published/draft） |
| featured | BOOLEAN | 是否精选 |
| created_at | TIMESTAMP | 创建时间 |
| updated_at | TIMESTAMP | 更新时间 |

### article_favorites 表

| 字段 | 类型 | 说明 |
|------|------|------|
| id | UUID | 主键 |
| user_id | UUID | 用户ID |
| article_id | UUID | 文章ID |
| created_at | TIMESTAMP | 收藏时间 |

### article_comments 表

| 字段 | 类型 | 说明 |
|------|------|------|
| id | UUID | 主键 |
| article_id | UUID | 文章ID |
| user_id | UUID | 用户ID |
| user_name | TEXT | 用户名 |
| content | TEXT | 评论内容 |
| likes | INTEGER | 点赞数 |
| parent_id | UUID | 父评论ID（回复功能） |
| created_at | TIMESTAMP | 评论时间 |

## 📸 替换真实图片

当前使用 Picsum 占位图，替换步骤：

```sql
-- 更新单篇文章图片
UPDATE culture_articles
SET cover_image = 'https://your-cdn.com/image.jpg'
WHERE slug = 'ancient-indigo-history';

-- 批量更新
UPDATE culture_articles
SET cover_image = REPLACE(
  cover_image, 
  'https://picsum.photos', 
  'https://your-cdn.com'
);
```

**图片建议**:
- 尺寸：1200x800px
- 格式：WebP 或 JPEG
- 大小：< 200KB
- 内容：与文章主题相关的蓝染照片

## 🆘 常见问题

### Q: 文章列表为空？
**A**: 检查：
1. SQL是否执行成功
2. Supabase连接是否正常
3. RLS策略是否正确配置

### Q: 图片无法显示？
**A**: 
- Picsum 服务偶尔不稳定
- 可以替换为本地图片或CDN

### Q: 分类筛选不工作？
**A**: 
- 检查URL参数是否正确
- 查看浏览器控制台错误

### Q: 如何添加新文章？
**A**:
```sql
INSERT INTO culture_articles (
  slug, title, excerpt, content, 
  cover_image, category, tags,

  '文章正文...',
  'https://image-url.jpg',
  '分类',
  ARRAY['标签1', '标签2'],
  10,
  '作者名'
);
```
