## 一、项目概况

### 1.1 项目定位

**世说蓝语**是一个融合传统蓝染文化与现代数字技术的综合性平台，致力于通过创新的方式传承和推广中国传统蓝染工艺。

### 1.2 核心价值

- 🎓 **教育传承**: 系统化的蓝染教学课程
- 🛍️ **文创电商**: 蓝染文创产品展示与销售
- 🎮 **互动体验**: 创新的蓝染创作游戏
- 🤖 **AI赋能**: 智能推荐与内容生成

## 二、技术架构

### 2.1 技术栈

**前端核心**
- Next.js 14.2 (App Router)
- React 18 + TypeScript 5+
- Tailwind CSS 4.1.9
- Radix UI 组件库

**后端/数据**
- Supabase (PostgreSQL + Auth + Storage)
- Next.js API Routes
- Row Level Security (RLS)

### 2.2 特色技术

**Canvas渲染引擎**
- 自主开发的高性能渲染系统
- 支持图层混合与实时预览
- 用于游戏中的蓝染作品绘制

**AI集成** (规划中)
- Vercel AI SDK + Google Gemini
- AI传记生成
- 智能推荐系统

### 4.4 智能评分系统

**AI评分算法**:
- **颜色深度**: 评估染色的丰富程度
- **图案平衡**: 评估布局的美观性
- **覆盖率**: 评估作品的完整度

**等级划分**:
- SSS (90-100分): 大师级作品
- SS (80-89分): 优秀作品
- S (70-79分): 良好作品
- A/B/C: 普通作品

**演示**:

![工坊评分](./工坊评分.png)

---

### 4.5 作品管理

**背包系统**:
- 查看我的作品
- 作品评分展示
- 上架到商店

**定价算法**:
```
最终价格 = 基础价 × 等级倍率 + 分数奖励

示例:
SSS级 95分 = 300 × 1.0 + 90 = 390 蓝草值
```
