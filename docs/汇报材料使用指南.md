### 步骤2: 制作PPT

#### 推荐工具
- **PowerPoint** (最常用)
- **Keynote** (Mac用户)
- **Google Slides** (在线协作)
- **WPS演示** (国产免费)

#### PPT模板建议
- **风格**: 现代简约、科技感
- **主色调**: 靛蓝色系 (#1f4e79, #5B7FA1, #C5D5E4)
- **辅助色**: 白色、浅灰色
- **字体**: 
  - 标题: 微软雅黑 Bold / 苹方黑体
  - 正文: 微软雅黑 Regular

#### 幻灯片制作要点

**封面 (第1页)**
```
【上方】
世说蓝语
蓝染文化传承平台

【中间】
中期检查汇报

【下方】
汇报人: XXX
日期: 2025年12月1日
版本: v1.1.2
进度: 90%

【背景】
靛蓝渐变 + 淡淡的蓝染纹理
```

**技术架构 (第4-7页)**
- 使用图标展示技术栈
- 流程图展示系统架构
- 数据表格要清晰易读

**功能演示 (第8-15页)**
- 每页1-2张大图
- 标注关键功能点
- 配合简短文字说明

**游戏系统 (第16-20页) ⭐重点**
- 第17页: 全屏展示工坊界面截图
- 嵌入30秒创作演示视频
- 评分界面要突出等级和分数

### 步骤3: 准备演示

#### 演示环境检查清单

```bash
□ 本地服务器启动正常
  npm run dev

□ 浏览器清空缓存
  Ctrl + Shift + Delete

□ 测试账号准备
  - 账号有购物车数据
  - 账号有收藏数据
  - 账号有订单记录
  - 账号有游戏作品

□ 网络连接稳定
  - Supabase连接正常
  - 图片加载正常

□ 备用方案准备
  - 截图已保存
  - 视频已录制
  - PPT中已嵌入
```

#### 演示设备准备

**投影仪/大屏**
- 分辨率: 1920x1080 或更高
- 连接方式: HDMI / 无线投屏
- 提前测试连接

**电脑配置**
- 关闭不必要的程序
- 关闭通知提醒
- 充满电或连接电源
- 准备翻页笔（可选）

**浏览器设置**
- 全屏模式 (F11)
- 隐藏书签栏
- 关闭其他标签页
- 准备好要演示的页面

### 步骤4: 彩排练习

#### 完整彩排流程

**第1遍: 熟悉内容 (30分钟)**
- 通读PPT内容
- 理解每页要点
- 记忆关键数据

**第2遍: 计时演练 (20分钟)**
- 按演示脚本练习
- 控制每部分时间
- 记录超时部分

**第3遍: 模拟演示 (15分钟)**
- 完整演示一遍
- 包含实际操作
- 控制总时间在15-20分钟

#### 时间分配建议

```
总时长: 15-20分钟

开场 (1-2分钟)
  - 自我介绍

  - 用户 (1分钟)
```

## 💡 演示技巧

### 口头表达

**开场白**
```
各位老师好，我是XXX。

今天向大家汇报"世说蓝语"蓝染文化传承平台的中期进展。

这是一个融合传统文化与现代技术的创新项目，
包含教学、商城、游戏、社交四大核心功能。

目前整体进度达到90%，让我为大家详细展示。
```

**过渡语**
- "接下来让我们看看..."
- "这部分是项目的亮点..."
- "特别值得一提的是..."
- "下面重点介绍..."

**结束语**
```
以上是我的汇报。

项目具备技术现代、功能完整、创新突出、性能优秀四大优势，
有信心打造成优秀的文化传承平台。

感谢各位老师的聆听，欢迎提问！
```

### 肢体语言

- ✅ 保持眼神交流
- ✅ 适当手势辅助
- ✅ 站姿端正自信
- ✅ 语速适中清晰
- ❌ 避免背对观众
- ❌ 避免频繁看稿
- ❌ 避免小动作

### 应急处理

**如果网络卡顿**
- 立即切换到录制视频
- "这里我们看一段提前录制的演示"

**如果功能出错**
- 使用截图说明
- "这是功能的正常效果"

**如果时间不够**
- 快速跳过次要功能
- 重点展示游戏系统

**如果被提问打断**
- 礼貌回应: "这个问题很好"
- 简短回答或承诺后续解答
- 继续汇报
