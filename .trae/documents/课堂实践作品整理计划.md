# 课堂实践作品整理计划

## 整理目标

根据用户要求，整理课堂实践作品文件，重点包括：

1. 不需要记录学号，只记录作者及作品名称
2. 商品产品图可以不提及作者名称
3. 保持视频与效果图的对应关系
4. 分类清晰，便于后续使用

## 整理方案

### 1. 目录结构设计

```
整理后课堂实践作品/
├── 视频教程/              # 所有视频文件
│   ├── 作者名/            # 按作者分类
│   │   ├── 作品名称-视频.mp4  # 统一命名格式
│   └── ...
├── 图案效果图/            # 所有效果图
│   ├── 作者名/            # 按作者分类
│   │   ├── 作品名称-效果图.png  # 统一命名格式
│   └── ...
└── 产品图片/              # 所有产品图
    ├── 产品类型/          # 按产品类型分类
    │   ├── 产品名称-序号-产品图.jpg  # 统一命名格式（不包含作者名）
    └── ...
```

### 2. 文件命名规则

#### 视频文件

* 格式：`作者名/作品名称-视频.mp4`

* 示例：`姚依然/蝴蝶纹样-视频.mp4`

#### 效果图

* 格式：`作者名/作品名称-效果图.png`

* 示例：`姚依然/蝴蝶纹样-效果图.png`

#### 产品图

* 格式：`产品类型/产品名称-序号-产品图.jpg`

* 示例：`帆布袋/吉祥如意锦鲤图-1-产品图.jpg`

### 3. 实施步骤

1. **创建基本目录结构**

   * 创建主要分类目录：视频教程、图案效果图、产品图片

   * 为每个作者创建子目录

   * 为不同产品类型创建子目录

2. **文件识别与分类**

   * 识别视频文件（.mp4/.MOV）

   * 识别效果图（.png）

   * 识别产品图（.jpg）

3. **文件处理流程**

   * 遍历每个学生目录

   * 提取作者名和作品名称

   * 复制并重命名文件到目标位置

   * 建立文件对应关系表

4. **产品类型分类**

   * 帆布袋

   * 方巾

   * 文化衫

   * 束口袋

   * 双肩包

   * 扇子

   * 纸巾盒

   * 其他

## 执行时间

预计需要10-15分钟完成所有文件的整理工作。

## 注意事项

1. 保留原始文件不变
2. 确保文件复制过程中不丢失任何数据
3. 对于特殊命名的文件，根据内容进行合理归类
4. 生成整理清单，便于后续查阅
