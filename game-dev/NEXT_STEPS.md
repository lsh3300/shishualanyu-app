## ✨ 游戏体验流程

### 在游戏主页 (`/game/hub`)
```
✅ 查看你的等级、经验、货币
✅ 点击 "创作工坊" 卡片
```

### 在游戏工坊 (`/game/workshop`)
```
1. 点击 "➕ 添加测试图层" 按钮（3-5次）
2. 点击 "完成创作" 按钮
3. 查看评分结果弹窗
   - 4个维度分数逐个显示
   - 总分和等级（SSS/SS/S/A/B/C）
   - 获得的奖励
   - 升级提示（如果升级）
4. 点击 "继续"
5. 观察顶部状态栏数据更新
6. 返回主页验证
```

### 多次创作
```
重复上述流程
→ 积累经验
→ 赚取货币
→ 升级！🎉
```
