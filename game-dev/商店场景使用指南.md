## 🎨 UI设计特点

### 1. 场景布局
```
视觉层次（从后到前）:
1. 背景图片（商店外景）
2. 画框展示区（中心）
3. 角色形象（左下角）
4. 对话气泡（角色上方）
5. 提示文字（右下角）
```

### 2. 动画效果

#### 进入动画
```
顺序：
0.3s → 角色从左侧滑入
0.4s → 画框1从上方落下
0.5s → 画框2从上方落下
0.6s → 画框3从上方落下
0.8s → 欢迎气泡弹出
1.0s → 底部提示淡入
```

#### 交互动画
```
画框：
- 悬停：scale(1.05)
- 选中：scale(1.1) + 黄色边框 + 光效
- 点击：触发选中状态

按钮：
- 悬停：颜色加深
- 点击：轻微缩放
```

### 3. 颜色方案
```
主色调：
- 背景：天空蓝渐变 (sky-50 to blue-50)
- 画框：琥珀棕色 (amber-800)
- 选中：金黄色 (yellow-400)
- 卡片：白色半透明

强调色：
- 今日收入：绿色 (green-500)
- 在售作品：蓝色 (blue-500)
- 访客数量：紫色 (purple-500)
```

## 🛠️ 技术实现

### 核心组件
```typescript
ShopScene.tsx
├─ 背景图片 (Next.js Image)
├─ 角色形象 (Next.js Image + Framer Motion)
├─ 画框组件 (FramePlaceholder)
│  ├─ 选中状态
│  ├─ 悬停效果
│  └─ 点击事件
└─ 欢迎气泡 (带动画)
```

### 状态管理
```typescript
状态：
- selectedFrame: number | null  // 当前选中的画框
- todayEarnings: number         // 今日收入
- listedCount: number           // 在售数量

未来扩展：
- listings: ShopListing[]       // 上架作品列表
- shopInfo: UserShop            // 商店信息
```
