# 📱 移动端使用指南

**适用于**: 手机、平板用户  
**版本**: v2.1

---

## 🎯 快速开始

### 第一次使用？

1. **打开浏览器** - Chrome、Safari、Firefox都支持
2. **访问网站** - 输入网址
3. **进入工坊** - 点击"蓝染创作"
4. **开始创作** - 跟着下面的步骤！

---

## 📖 基础操作

### 1️⃣ 选择图案

```
┌─────────────────────────┐
│ ❄️ 🌸 🌀 ⭐ ◇ 〰️ ✿ ✚  │ ← 左右滑动
└─────────────────────────┘
```

**怎么做**：
- 👆 **左右滑动** 浏览所有图案（13个）
- 🎯 **点击图案** 选中它
- ✨ 选中的图案会有蓝色边框
- 📍 自动滚动到中心

**提示**：
- 👈👉 看到箭头提示表示可以继续滑动
- 🔵 蓝色边框 = 已选中

---

### 2️⃣ 放置图案

```
┌─────────────┐
│             │
│      ✓      │ ← 点击画布
│             │
└─────────────┘
```

**怎么做**：
1. 选中图案后
2. **点击画布**任意位置
3. 图案就出现了！

**提示**：
- 图案中心会在你点击的位置
- 可以添加多个图案

---

### 3️⃣ 移动图案

```
图案 → 按住拖动 → 新位置
```

**怎么做**：
- 👆 **点击并按住**图案
- 👉 **拖动**到想要的位置
- 🎯 **松开手指**完成移动

---

### 4️⃣ 使用操作菜单

```
           ┌──┐
           │📋│ ← 点击打开
           └──┘
        ┌──┐
        │⟲ │ ← 撤销
        ├──┤
        │⟳ │ ← 重做
        ├──┤
        │📋│ ← 复制
        ├──┤
        │🗑️│ ← 删除
        └──┘
```

**怎么打开**：
- 点击右下角的**圆形按钮**（菜单图标）
- 菜单展开显示所有操作

**怎么关闭**：
- 点击任意操作按钮（自动关闭）
- 点击屏幕其他地方
- 再次点击菜单按钮

---

## 🎨 进阶操作

### 5️⃣ 调整图案属性

**打开属性面板**：
1. 点击右下角菜单按钮
2. 选择"属性"按钮
3. 属性面板出现

**可以调整**：
- 🔍 **位置** - 精确设置X/Y坐标
- 📏 **缩放** - 20%到300%
- 🔄 **旋转** - 0°到360°
- 👁️ **不透明度** - 10%到100%
- 💧 **染色深度** - 颜色深浅

**快捷预设**（一键设置）：
- 🌸 **精致小花** - 小巧可爱
- 🌟 **醒目主题** - 大而显眼
- ⚪ **标准样式** - 默认设置
- 💫 **深邃神秘** - 深色浓郁

---

### 6️⃣ 管理图层

**打开图层面板**：
1. 点击菜单按钮
2. 选择"图层"
3. 看到所有图层列表

**可以做什么**：
- 👁️ **显示/隐藏** - 点击眼睛图标
- 🔒 **锁定/解锁** - 点击锁图标
- ⬆️ **上移图层** - 改变堆叠顺序
- ⬇️ **下移图层** - 改变堆叠顺序
- 📋 **复制图层** - 快速复制
- 🗑️ **删除图层** - 移除不要的

---

### 7️⃣ 撤销和重做

**撤销（Undo）**：
- 点击菜单 → "撤销"按钮
- 回退上一步操作

**重做（Redo）**：
- 点击菜单 → "重做"按钮
- 前进到下一步

**提示**：
- 可以无限次撤销！
- 操作失误不用怕

---

### 8️⃣ 复制图案

**快速复制**：
1. 点击要复制的图案（选中它）
2. 打开菜单
3. 点击"复制"按钮
4. 新图案会出现在旁边

**用途**：
- 创建对称图案
- 批量添加相似图案
- 快速布局

---

### 9️⃣ 删除图案

**删除单个**：
1. 选中要删除的图案
2. 打开菜单
3. 点击"删除"

**清空全部**：
1. 打开菜单
2. 点击"清空"
3. 确认删除

**注意**：
- 删除前会提示确认
- 可以用撤销恢复

## ❓ 常见问题

### Q1: 滑动图案时误触怎么办？
**A**: 慢慢滑动，不要太快。选中了不想要的图案就点击其他图案取消选择。

### Q2: 图案放置位置不准确？
**A**: 使用属性面板精确输入坐标，或者放大查看。

### Q3: 找不到操作按钮？
**A**: 看屏幕右下角，圆形的菜单按钮，点击它展开所有操作。

### Q4: 怎么保存作品？
**A**: 完成创作后，点击"完成创作"按钮，系统会自动保存和评分。

### Q5: 误删了图案怎么办？
**A**: 立即点击菜单 → 撤销，图案就回来了！

### Q6: 为什么有些功能是灰色的？
**A**: 灰色表示当前不可用，比如：
- 没有历史记录时，撤销/重做是灰色
- 没选中图案时，复制/删除是灰色

### Q7: 手机屏幕小，图案看不清？
**A**: 
- 双指缩放查看（未来功能）
- 使用属性面板的快捷预设
- 横屏使用可以看更清楚

## 🎯 最佳实践

### ✅ 推荐做法
- 📱 竖屏使用（最佳体验）
- 🎨 先选图案，再放置
- 💾 经常保存（点完成创作）
- 🎭 多尝试不同组合
- 📐 使用属性面板精确调整

### ❌ 避免这样
- 不要太快滑动图案（可能误触）
- 不要一次添加太多图案（会卡）
- 不要忘记保存作品
- 不要怕犯错（有撤销功能）

---

## 🚀 性能建议

### 如果感觉卡顿
1. 减少图层数量（删除一些）
2. 关闭不需要的面板
3. 清理浏览器缓存
4. 重启应用

### 推荐配置
- 📱 **内存**: 2GB以上
- 📶 **网络**: WiFi或4G
- 🔋 **电量**: 50%以上
- 🌐 **浏览器**: Chrome/Safari最新版
