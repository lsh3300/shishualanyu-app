# 📐 新布局使用指南

**更新日期**: 2025-11-30  
**新功能**: 侧边栏折叠 + 完美响应式

---

## 🆕 新增功能

### 1. 侧边栏折叠按钮（PC端）

在工具栏右上角，你会看到两个新按钮：

```
┌─────────────────────────────────┐
│ 🎨 蓝染创作工坊    [📋][⚙️] ⏮️ ⏭️ 🗑️ │
└─────────────────────────────────┘
             ↑    ↑
         图层管理  属性面板
```

**使用方法**：
- 点击 **[📋]** - 显示/隐藏图层管理面板
- 点击 **[⚙️]** - 显示/隐藏属性面板
- 蓝色高亮 = 面板已展开
- 灰色 = 面板已折叠

**何时使用**：
- 需要更大的画布空间
- 专注于创作不想被打扰
- 屏幕空间有限时

---

### 2. 响应式布局

**手机/小平板（<1024px）**：
```
┌─────────────┐
│   标题      │
├─────────────┤
│   画布      │
│  (自适应)   │
├─────────────┤
│ 图案选择器  │
└─────────────┘

面板通过右下角浮动按钮打开
```

**大屏幕（≥1024px）**：
```
┌────┬───────┬────┐
│图层│ 画布  │属性│
│管理│(600px)│面板│
└────┴───────┴────┘
│   图案选择器    │
└────────────────┘

可以点击折叠按钮隐藏侧边栏
```

---

### 3. 面板滚动

**现在所有面板都支持滚动！**

- 图层管理 - 添加很多图层也不怕
- 属性面板 - 所有选项都能看到
- 自定义滚动条 - 更美观

**如何滚动**：
- 鼠标滚轮
- 触摸滑动
- 拖动滚动条

## 📱 不同设备使用指南

### 手机使用

1. **查看图层**：
   - 点击右下角菜单按钮
   - 选择"图层"
   - 图层面板从下方弹出
   - 操作完成后点击其他地方关闭

2. **调整属性**：
   - 先选中画布上的图案
   - 点击右下角菜单
   - 选择"属性"
   - 属性面板从下方弹出

3. **专注创作**：
   - 关闭所有面板
   - 画布占满整个屏幕
   - 更好的创作体验

### 平板使用

1. **竖屏模式**：
   - 垂直堆叠布局
   - 滚动查看所有内容
   - 和手机类似

2. **横屏模式**：
   - 如果宽度 ≥1024px
   - 自动切换到水平布局
   - 侧边栏并排显示

### PC使用

1. **全功能模式**（默认）：
   - 图层管理在左侧
   - 画布在中间
   - 属性面板在右侧
   - 所有功能一目了然

2. **专注模式**：
   - 点击折叠按钮隐藏侧边栏
   - 画布占据更多空间
   - 需要时再打开面板

3. **自定义模式**：
   - 只打开图层管理
   - 只打开属性面板
   - 或都不打开

## ❓ 常见问题

### Q1: 侧边栏不见了怎么办？
**A**: 点击工具栏的折叠按钮（[📋] 或 [⚙️]），面板就会重新出现。

### Q2: 移动端怎么打开面板？
**A**: 点击右下角的圆形菜单按钮，选择"图层"或"属性"。

### Q3: 面板内容看不全？
**A**: 现在所有面板都支持滚动，用滚轮或手指滑动即可。

### Q4: 缩放浏览器后布局混乱？
**A**: 新版本已修复，任意缩放都会自动适配。

### Q5: 能否记住我的折叠状态？
**A**: 目前还不支持，每次刷新会恢复默认状态（未来会添加）。

### Q6: PC端能否拖拽调整侧边栏宽度？
**A**: 目前宽度固定（320px或384px），未来版本会支持拖拽调整。
