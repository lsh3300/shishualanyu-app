## 项目概述
世说蓝语 (SSL) 应用是一个集文化内容展示、商品销售、学习互动于一体的综合平台，致力于传承和推广传统靛蓝文化。

## 功能模块
- 文化文章展示与收藏
- 靛蓝产品销售与购物车
- 在线学习课程
- 互动游戏体验
- 用户收藏系统
- 消息通知中心

## 技术架构
- **前端框架**: Next.js 14
- **数据库**: Supabase (PostgreSQL)
- **存储**: AWS S3
- **认证**: Supabase Auth
- **UI组件**: Tailwind CSS + 自定义组件

## 数据库结构
主要数据表：
- users - 用户信息表
- products - 产品信息表
- favorites - 收藏记录表
- culture_articles - 文化文章表
- orders - 订单信息表

## 项目结构
```
app/
├── api/          # API路由
├── auth/         # 认证相关页面
├── cart/         # 购物车功能
├── culture/      # 文化文章功能
├── profile/      # 用户个人中心
├── store/        # 产品商店
└── indigo-game/  # 互动游戏
components/       # UI组件
lib/              # 工具函数
hooks/            # React Hooks
```

## 快速恢复步骤
1. 克隆项目代码：`git clone <repository-url>`
2. 切换到v1.1.1版本：`git checkout v1.1.1`
3. 安装依赖：`npm install` 或 `pnpm install`
4. 配置环境变量：复制 `.env.example` 为 `.env.local` 并填写相应配置
5. 初始化数据库：运行数据库迁移脚本
6. 启动开发服务器：`npm run dev`

## 注意事项
- 确保使用正确的Supabase项目配置
- 环境变量中敏感信息请妥善保管
- 数据库迁移前请备份现有数据
- 生产环境部署前请执行全面测试
