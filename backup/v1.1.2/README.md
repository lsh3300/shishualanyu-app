## 项目概述
世说蓝语（sslyapp）是一个集成文化内容、产品展示、学习课程和用户互动功能的综合性应用平台。

## 功能模块
- 文化文章展示与收藏
- 产品浏览与购物车系统
- 在线课程学习
- 用户互动（点赞、评论）
- 个人中心管理

## 技术架构
- **前端**: Next.js 14, React 18, TypeScript
- **数据库**: Supabase
- **状态管理**: React Hooks, Context API
- **UI 组件**: Tailwind CSS, shadcn/ui
- **API**: RESTful API

## 数据库结构
- users - 用户信息
- products - 产品信息
- courses - 课程信息
- culture_articles - 文化文章
- favorites - 收藏记录
- likes - 点赞记录
- comments - 评论记录
- cart_items - 购物车商品

## 项目结构

## 快速恢复步骤
1. 检出 v1.1.2 标签: `git checkout v1.1.2`
2. 安装依赖: `npm install` 或 `yarn install` 或 `pnpm install`
3. 配置环境变量: 复制 `.env.example` 为 `.env.local` 并填充相关配置
4. 初始化数据库: 根据需要运行数据库迁移脚本
5. 启动开发服务器: `npm run dev`

## 注意事项
- 请确保使用正确的 Node.js 版本
- 环境变量配置必须完整且正确
- 数据库迁移前请备份现有数据
